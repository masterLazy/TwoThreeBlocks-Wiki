## 本 Wiki 是如何工作的

我们使用 **[Markdown](https://markdown.com.cn/)** 来编写文档。Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。下面是一个例子。

这就是本文部分 Markdown 看起来的样子。这些 Markdown 被称为源文件，它们将经过一系列工具的处理，最终构建成这个 Wiki 网页。这些网页通过 GitHub Pages 托管在互联网上，因而你可以在任何地方查看本 Wiki。

总结：要编写 Wiki 文档，你主要的任务就是编写 Markdown。

更具体地，本 Wiki 使用 Docusaurus 来生成网站，这是一个由 Facebook 开发的静态网页构建工具。

如何添加地点 Wiki 条目​

本 Wiki 的大部分条目都是地点条目。地点条目至少包含以下信息：

  1. 基本信息：地点的名称、建造者、坐标
  2. 一张游戏内截图

如果加入以下信息，条目会更完善：

  1. 关于地点的描述性文本，比如产量、参考的视频教程、产品、目的等
  2. 使用方法（对于机器）

如果你不希望自己编写 Markdown，可以将相关材料提交给 msterLazy（你可以在二三方块人交流会的 QQ 群中找到他）。下面将介绍如何提供最基本的信息。

如何查看坐标​

在游戏内按下键盘上的 F3，屏幕左侧便会显示你当前的坐标。从左至右分别是 X 坐标、Y 坐标、Z 坐标。请注意 Y 轴垂直于地面，这与数学上惯用的空间直角坐标系不同。

红框内的就是玩家当前坐标；这位于屏幕左侧

如何截图​

推荐的截图步骤如下：

  1. 按下键盘上的 F1，隐藏准星、物品栏、聊天框、计分板等。你稍后可以再次按下 F1 恢复。
  2. 按下键盘上的 F2 截图。
