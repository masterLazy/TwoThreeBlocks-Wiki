P8110 [Cnoi2021] 矩阵

难度算法s日期题目链接
普及+/提高快速幂2025-09-24https://luogu.com.cn/problem/P8110

看似矩阵快速幂，实则推式子 + 普通快速幂。

题意简述​

给定两个长度为 nnn 的数列 {a1,a2,…,an},{b1,b2,…,bn}\{a_1,a_2,\dots,a_n\},\{b_1,b_2,\dots,b_n\}{a1​,a2​,…,an​},{b1​,b2​,…,bn​}，n×nn\times nn×n 的矩阵 AAA 满足 Aij=ai×bjA_{ij}=a_i\times b_jAij​=ai​×bj​。

求：(∑i=1n∑j=1nAijk) mod M(\sum_{i=1}^n\sum_{j=1}^nA_{ij}^k)\bmod M(∑i=1n​∑j=1n​Aijk​)modM，其中 M=998244353M=998244353M=998244353。

范围：1≤n≤1051\leq n\leq10^51≤n≤105；0≤k≤M0\leq k\leq M0≤k≤M；ai,bi∈Ra_i,b_i\in\mathbb{R}ai​,bi​∈R，∣ai∣,∣bi∣≤109|a_i|,|b_i|\leq10^9∣ai​∣,∣bi​∣≤109；

（M=998244353≈109M=998244353\approx10^9M=998244353≈109）

本题中 A0A^0A0 为单位矩阵。

思路​

如果用常规矩阵快速幂，求一次矩阵乘法 O(n2)O(n^2)O(n2)，总共 O(nlog⁡n2)O(n\log n^2)O(nlogn2)，显然不行。

考虑推式子。

Aij2=Aij×Aij=∑k=1naibk×akbj=aibj∑k=1nakbkA^{2}_{ij}=A_{ij}\times A_{ij}=\sum_{k=1}^n a_ib_k\times a_kb_j=a_ib_j\sum_{k=1}^na_kb_kAij2​=Aij​×Aij​=∑k=1n​ai​bk​×ak​bj​=ai​bj​∑k=1n​ak​bk​；

不妨记 σ=∑k=1nakbk\sigma=\sum_{k=1}^na_kb_kσ=∑k=1n​ak​bk​，

则 Aij2=aibjσA^{2}_{ij}=a_ib_j\sigmaAij2​=ai​bj​σ。

类似地，Aij4=Aij2×Aij2=∑k=1naibkσ×akbjσ=aibjσ2∑k=1naibj=aibjσ3A^{4}_{ij}=A^{2}_{ij}\times A^{2}_{ij}=\sum_{k=1}^na_ib_k\sigma\times a_kb_j\sigma=a_ib_j\sigma^2\sum_{k=1}^na_ib_j=a_ib_j\sigma^3Aij4​=Aij2​×Aij2​=∑k=1n​ai​bk​σ×ak​bj​σ=ai​bj​σ2∑k=1n​ai​bj​=ai​bj​σ3。

不难发现，Aijk=aibjσk−1A^k_{ij}=a_ib_j\sigma^{k-1}Aijk​=ai​bj​σk−1。

故：

∑i=1n∑j=1nAijk=∑i=1n∑j=1naibjσk−1=(∑i=1nai)⋅(∑j=1nbj)⋅(∑i=1naibi)k−1.\sum_{i=1}^n\sum_{j=1}^nA_{ij}^k=\sum_{i=1}^n\sum_{j=1}^na_ib_j\sigma^{k-1}=(\sum_{i=1}^na_i)\cdot(\sum_{j=1}^nb_j)\cdot(\sum_{i=1}^na_ib_i)^{k-1}.i=1∑n​j=1∑n​Aijk​=i=1∑n​j=1∑n​ai​bj​σk−1=(i=1∑n​ai​)⋅(j=1∑n​bj​)⋅(i=1∑n​ai​bi​)k−1.

所以我们只需要用 O(n)O(n)O(n) 的时间统计三个 ∑\sum∑ 的值，然后对 σ\sigmaσ 用快速幂，总计 O(n+log⁡n)=O(n)O(n+\log n)=O(n)O(n+logn)=O(n)，可以通过本题。
