最大公因数 GCD​

gcd⁡(a,b)\gcd(a,b)gcd(a,b) 表示 a,ba,ba,b 的最大公因数。

求 gcd⁡\gcdgcd 可以用 std::__gcd()。如果要自己搓，常用欧几里得算法，又叫辗转相除法，即 gcd⁡(a,b)=gcd⁡(b,amod  b)\gcd(a,b)=\gcd(b,a\mod b)gcd(a,b)=gcd(b,amodb)。

int gcd(int a, int b) {
    return b ? gcd(b, a % b) : a;
}

gcd⁡\gcdgcd 的性质：

  • 结合律：gcd⁡(gcd⁡(a,b),c)=gcd⁡(a,gcd⁡(b,c))=gcd⁡(a,b,c)\gcd(\gcd(a,b),c)=\gcd(a,\gcd(b,c))=\gcd(a,b,c)gcd(gcd(a,b),c)=gcd(a,gcd(b,c))=gcd(a,b,c)。交换律当然也有。

  • gcd⁡(ma,mb)=mgcd⁡(a,b)\gcd(ma,mb)=m\gcd(a,b)gcd(ma,mb)=mgcd(a,b)。

裴蜀定理​

裴蜀定理：对于已知的整数 a,ba,ba,b，存在整数 x,yx,yx,y 使得 ax+by=gcd⁡(a,b)ax+by=\gcd(a,b)ax+by=gcd(a,b)。

推论：

  • a⊥ba\perp ba⊥b 当且仅当 ax+by=1.ax+by=1.ax+by=1.

  • ax+by=dax+by=dax+by=d 有解（a,ba,ba,b 已知），当且仅当 d=kgcd⁡(a,b).d=k\gcd(a,b).d=kgcd(a,b).（二元线性丢番图方程）

裴蜀定理很好证明：

  • 对任意 x,yx,yx,y，xa+by=dxa+by=dxa+by=d，则 d=kgcd⁡(a,b)d=k\gcd(a,b)d=kgcd(a,b)，kkk 是整数。

  • 为什么？因为 gcd⁡(a,b)(xr1+yr2)=d\gcd(a,b)(xr_1+yr_2)=dgcd(a,b)(xr1​+yr2​)=d。容易看出：最小的 ddd 是 gcd⁡(a,b)\gcd(a,b)gcd(a,b)，当且仅当 xr1+yr2=1xr_1+yr_2=1xr1​+yr2​=1。（这里 r1,r2r_1,r_2r1​,r2​ 是互素的。）
