刷沙机

主要建造者：		msterLazy, lemoncola_87
末地（主体）：		0 ~ 0
主世界（收集）：		-120 +89 +100

可以刷一切重力方块，包括：沙子、红沙、沙砾、混凝土粉末，甚至还有铁砧、龙蛋。

参考 BV1rP411W7DG 设计。

使用方法​

  • 来到末地的返回传送门旁，找到机器。

  • 把活塞上的原有的重力方块换成你想要刷的重力方块。

  • 松开上图中指示的拉杆，机器就会开始运行。

危险

机器运行时会有 TNT 爆炸，请保持安全距离。

  • 要让机器正常运行，至少需要一人在末地主岛挂机。刷取的重力方块将在主世界出生点旁被收集。
  • 离开末地前请务必关闭刷沙机，等待所有 TNT 爆炸结束后返回主世界。刷沙机的效率很惊人，一次挂机建议在 10 分钟以内。

收集装置​

其实就在出生点旁。除了铁砧和龙蛋，所有的重力方块都做了收集系统。你可以通过水电梯来到这个平台。
