袭击塔

主要建造者：		sin_zri, LeaDer_LD
下界：			-2 ~ +200
主世界：			+3 ~ +1617

生产绿宝石、不死图腾、不详旗帜、鞍等。

使用方法​

important

离开袭击塔之前一定要确保身上的不祥之兆效果已经清空，否则可能引起悲剧。

  1. 前往袭击塔附近的掠夺者前哨站，坐标在下面。通过击杀袭击队长，获得至少一个不详之瓶，然后返回袭击塔。如果你不想用坐标，也可以先沿着铁路的方向找到大陆，然后在面向大陆的左边找到高耸的圆石柱，顺着圆石柱所指的方向前往前哨站。
掠夺者前哨站坐标
-8 +63 +1272
  1. 跟随下面图示的指引，来到挂机平台。在这里喝下不详之瓶，然后会触发袭击。你在这里给怪物 “削脚”，击退袭击。

  1. 一轮袭击结束后，你可以跳入身后的洞中（你转身后的左边 / 图中右边小图上面没有水的那个），你会降落在一个村民旁边（不用担心，下面有水），然后通过气泡柱回到挂机平台，这样你就能触发新一轮的袭击。
  2. 确保身上的不祥之兆效果已经清空，然后你可以离开袭击塔。
