包含 34 个熔炉。

## 使用方法

1. 打开刷地毯机。（拉下中间的拉杆）
2. 让漏斗矿车吸入你想烧制的物品，然后让矿车开始运行。
3. 待熔炉全部熄灭后，在中间的大箱子拿出成品。

使用完后如果所有熔炉（和其上的漏斗）中的燃料都已满，可以关闭刷地毯机。
